Fourier-Argand Filter

=====================
This code is a Matlab implementation of the Fourier-Argand filter algorithm.

Authors: 
	Tianle Zhao and Thierry Blu

Usage:
	Follow the steps below:
		1. Change MATLAB "present working directory (pwd)" to where this script 
			locates.
		2. Run the demo scripts.

Tips:
	Click on the figure to see an alternating comparison when there're multiple
	images.
